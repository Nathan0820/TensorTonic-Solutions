# TensorTonic Solutions

A collection of my **from-scratch** machine learning and deep learning implementations from [TensorTonic](https://tensortonic.com), synced here for version control and review.

## What is TensorTonic?
TensorTonic is a platform of hands-on exercises where you implement core ML concepts and algorithms yourself (rather than relying only on high-level libraries).

## What’s in this repository
- **Algorithm implementations** (e.g., optimization, regression/classification, neural nets)
- **Exercise / problem solutions** pulled from TensorTonic
- **Incremental improvements** over time as I revisit and refine approaches

> Note: The exact set of folders/files may change as TensorTonic adds new challenges and as my progress updates.

## How to use
1. Browse the repository folders to find a topic/algorithm.
2. Open the relevant source files to see the implementation and any notes.
3. (Optional) Clone the repo to run locally:

```bash
git clone https://github.com/Nathan0820/TensorTonic-Solutions.git
cd TensorTonic-Solutions
```

## Goals
- Practice implementing ML/DL fundamentals without “black-boxing” the math.
- Keep an organized, searchable record of progress.
- Make it easy to revisit solutions and compare iterations.

## Disclaimer
These are **personal learning solutions** and may not be the most optimized or production-ready implementations.
